# Abhimat Dangi Edits

A professional video editing portfolio website showcasing my short-form and long-form editing work, client results, and services.

## Live Website
🌐 https://abhimatdangi.github.io/Abhimat-Dangi-Edits/

## Overview
This portfolio website highlights my video editing services, previous work, and client testimonials.  
It is designed to clearly present my editing style, pricing, and contact options for potential clients.

## Features

- 🎬 Video showcase with YouTube integration  
- 💬 Real client testimonials  
- 🌙 Dark themed modern UI  
- 💰 Transparent pricing section  
- 📩 Contact & booking section  
- 📱 Fully responsive design (mobile, tablet, desktop)

## Tech Stack

This project is built using:

- **React**
- **TypeScript**
- **Vite**
- **Tailwind CSS**
- **shadcn/ui**

## Installation (Run Locally)

Clone the repository and install dependencies.

```bash
git clone https://github.com/abhimatdangi/Abhimat-Dangi-Edits.git
cd Abhimat-Dangi-Edits
npm install
npm run dev
